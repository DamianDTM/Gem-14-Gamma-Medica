# Herramientas y Servicios Configurados

- **Control de Versiones y Hosting:**
  - `gh cli` / `github_mcp`: Para manejo de repositorios y despliegues.
  - `vercel cli` / Plataforma Vercel: Host principal de la web.

- **Diseño y UI:**
  - `StitchMCP`: Usado para diseñar las vistas y UI del proyecto (migrando desde el look and feel de Elementor).
  - `browser_subagent`: Para validar visualmente que el render sea correcto.
